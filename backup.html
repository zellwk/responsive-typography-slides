Changing Font-size

Changing Font-size when screen width changes

html {
  font-size: 16px;
  @media (min-width: 600px) {
    font-size: 18px;
  }
  @media (min-width: 1000px) {
    font-size: 20px;
  }
}

Changing Font-size and line-height

html {
  font-size: 16px;
  line-height: 1.4;
  @media (min-width: 600px) {
    font-size: 18px;
  }
  @media (min-width: 1000px) {
    font-size: 20px;
    line-height: 1.5;
  }
}

Use Relative Units

  • Em
  • Rem

Em

An EM is a unit of typography, equal to the currently specified point-size

Em

h1 { font-size: 20px } /* 1em = 20px */
p { font-size: 16px } /* 1em = 16px */

Using Em on font-size property

html { font-size: 16px }
h1 { font-size: 2em } /* 16px * 2 = 32px */

Using Em on other properties

h1 {
  font-size: 2em; /* 1em = 16px */
  margin-bottom: 1em; /* 1em = 32px */
}

p {
  font-size: 1em; /* 1em = 16px */
  margin-bottom: 1em; /* 1em = 16px */
}

Rem

An REM is a unit of typography, equal to the root font-size

Rem

html { font-size: 16px } /* 1rem === 16px, always */

Using Rem on other properties

h1 {
  font-size: 2rem; /* 1rem = 16px */
  margin-bottom: 1rem; /* 1rem = 16px */
}

p {
  font-size: 1rem; /* 1rem = 16px */
  margin-bottom: 1rem; /* 1rem = 16px */
}

Introducing Typi

(The fix for modular scale madness)
(Also the fix for "What if my designer doesn't follow modular scale???")

https://github.com/zellwk/typi

Using Typi (body text)

\$breakpoints: (
  small: 600px,
  large: 1000px
);
              
\$typi: (
  null: (16px, 1.4),
  small: 18px,
  large: (20px, 1.5)
);

Using Typi (body text)

// Sass input
html {
  @include typi-base();
}













 /* CSS Output */
html {
  font-size: 100%;
  line-height: 1.4;
}

@media all and (min-width: 600px) {
  html {
    font-size: 112.5%;
  }
}

@media all and (min-width: 1000px) {
  html {
    font-size: 125%;
    line-height: 1.5;
  }
}

Using Typi (Other elements)

// Scss input
\$h1-map: (
  null: (2.369em, 1.2),
  large: (3.157em, 1.3)
);

\$h2-map: (
  null: (1.777em, 1.2),
  large: 2.369em
);

h1 { @include typi(\$h1-map); }
h2 { @include typi(\$h2-map); }




/* CSS Output */
h1 {
  font-size: 2.369rem;
  line-height: 1.2;
}

@media all and (min-width: 1000px) {

}

}

  }
}
